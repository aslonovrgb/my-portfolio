Nuvora Studio — ready-to-upload package

Upload these items together to hosting:
- index.html
- assets/ folder

Do not rename the assets folder, because index.html links to:
- assets/css/style.css
- assets/js/main.js
- assets/img/favicon.svg
- assets/img/og-preview.svg

Included:
- Fixed Mobile Apps / Flutter icons with reliable CSS icon fallback.
- Live preview browser cards for weel.uz and crestlink.net.
- Expanded live-preview modal with desktop/mobile preview modes.
- Redesigned process section.
- Updated About text to present Nuvora as a team.
- Contact form helpers: quick project format chips and local draft autosave.

Note: some external sites can block iframe embedding through their own security settings. In that case, the portfolio still shows the preview card and the visitor can open the site via the button.
