# Nuvora Studio website

Полноценная версия сайта под бренд **Nuvora Studio**.

## Что внутри

- `index.html` — главная страница
- `assets/css/style.css` — стили
- `assets/js/main.js` — интерактивность
- `assets/img/favicon.svg` — иконка
- `assets/img/og-preview.svg` — preview для соцсетей
- `assets/docs/Nuvora-Studio-Overview.txt` — временный файл вместо PDF-презентации/CV

## Как открыть

Открой `index.html` внутри папки проекта. Не вытаскивай его отдельно, иначе браузер не найдет `assets`.

## Перед публикацией

1. Замени `Nuvora-Studio-Overview.txt` на реальный PDF.
2. Обнови домен в Open Graph meta-тегах.
3. Замени concept/demo кейсы на реальные проекты, когда они будут готовы.
4. Проверь Formspree ID в форме.
